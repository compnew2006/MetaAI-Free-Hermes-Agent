# Contributing to Meta AI Python SDK

We welcome contributions from the community! Here is how you can help:

## Code of Conduct

Please be respectful and ethical in all your interactions within the project.

## How Can I Contribute?

### 🐛 Reporting Bugs
- Search existing issues before creating a new one.
- Describe the bug clearly with steps to reproduce and system details.

### 💡 Proposing Features
- Describe the feature use case and implementation suggestion in detail.

### 🔧 Submitting Pull Requests
1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/cool-feature`
3. Commit your changes: `git commit -m 'feat: add cool feature'`
4. Push to the branch: `git push origin feature/cool-feature`
5. Open a Pull Request.

## Development Setup

### Install from source
```bash
git clone https://github.com/compnew2006/meta.ai-sdk.git
cd meta.ai-sdk
pip install -e ".[api]"
```
